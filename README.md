## 🚀 Overview

LangDetect Pro is a modern web application that identifies the language of user-provided text using Natural Language Processing.

It provides primary language detection along with confidence scores in a clean and interactive Streamlit dashboard.

---

## ✨ Features

✅ Detects text language instantly
✅ Displays confidence percentages
✅ Supports multiple languages
✅ Modern responsive UI
✅ Fast and lightweight performance
✅ Built with Python and NLP
